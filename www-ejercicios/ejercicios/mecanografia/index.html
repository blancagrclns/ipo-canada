Guía del Test de Mecanografía

Funcionamiento Básico

El test mide tu velocidad de escritura mostrando palabras que debes teclear correctamente. El temporizador cuenta los segundos transcurridos.

  • Haz clic en Comenzar Test para iniciar
  • Escribe la palabra mostrada y presiona Enter para validar
  • El test finaliza al alcanzar el número de palabras o tiempo establecido

Configuración

  • Puedes establecer un número máximo de palabras (opcional)
  • Puedes fijar un tiempo máximo en segundos (opcional)
  • Activa el Modo Bilingüe para practicar traducción entre idiomas

Niveles de Dificultad

Cambia la dificultad en tiempo real presionando teclas numéricas (0-4) durante el test:

Tecla Nivel Descripción
0 Default Palabras mixtas de uso común
1 Fácil Monosílabos (sol, pan, mar...)
2 Medio Palabras agudas/llanas (reloj, hotel...)
3 Difícil Palabras esdrújulas (teléfono, pájaro...)
4 Muy Difícil Palabras sobresdrújulas/acentuadas (dígamelo...)

Modo Bilingüe

Practica la traducción entre idiomas:

  • Verás una palabra en un idioma (por ejemplo, en inglés)
  • Deberás escribir su traducción en el otro idioma (por ejemplo, en español)
  • Puedes configurar la dirección de la traducción (inglés→español o español→inglés)
  • El modo bilingüe solo se puede activar/desactivar antes de comenzar el test

Controles Adicionales

  • El botón Pausar te permite detener temporalmente el test
  • El botón Fin finaliza el test y muestra tus estadísticas
  • Puedes cambiar entre tema claro y oscuro con el botón Tema Oscuro
