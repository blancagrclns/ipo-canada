Instrucciones del Test

Escribe las palabras mostradas lo más rápido posible. El temporizador cuenta segundos transcurridos.

Puedes pausar/reanudar el test en cualquier momento. También puedes fijar un número de palabras o un tiempo máximo antes de comenzar.

Dificultad dinámica: Presiona teclas numéricas (0-4) para cambiar dificultad en tiempo real sin detener el test:

  • 0: Default (palabras mixtas)
  • 1: Fácil (monosílabos)
  • 2: Media (agudas/llanas)
  • 3: Difícil (esdrújulas)
  • 4: Muy difícil (sobresdrújulas/acentos)

Variantes disponibles: Pausa temporal, límite de palabras, tiempo máximo, y esta ayuda.
