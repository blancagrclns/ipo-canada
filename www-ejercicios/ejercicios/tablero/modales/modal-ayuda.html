Ayuda del Tablero

Objetivo del Juego

Según el modo de juego seleccionado:

  • Modo Normal: Intercambia las fichas para conseguir que cada fila del tablero contenga fichas del mismo color.
  • Modo Dos Tableros: Mueve todas las fichas del tablero origen al tablero destino, organizándolas para que cada fila tenga fichas del mismo color.
  • Modo Rompecabezas: Intercambia las piezas para formar correctamente la imagen original.

Cómo Jugar

  1. Configura el tamaño del tablero, características de las fichas y el modo de juego.
  2. Pulsa Comenzar Juego para iniciar una partida.
  3. Haz clic y arrastra una ficha para intercambiarla o moverla según el modo.
  4. Puedes Pausar el juego en cualquier momento.
  5. El botón Detener Juego finaliza la partida actual.

Configuración

  • Tamaño de la ficha: Cambia las dimensiones de las fichas (pequeña, mediana o grande).
  • Tamaño del tablero: Define el número de filas y columnas (de 2 a 7).
  • Forma de fichas: Elige entre cuadrados, círculos o triángulos.
  • Modo de juego: Selecciona entre Normal, Dos Tableros o Rompecabezas.
