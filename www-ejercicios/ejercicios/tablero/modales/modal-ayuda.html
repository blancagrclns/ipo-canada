Ayuda del Tablero

Objetivo del Juego

Intercambia las fichas mediante arrastre para conseguir que cada fila del tablero contenga fichas del mismo color. El juego se completa cuando todas las filas tienen un único color.

Cómo Jugar

  1. Configura el tamaño del tablero y las características de las fichas.
  2. Pulsa Comenzar Juego para iniciar una partida.
  3. Haz clic y arrastra una ficha hacia otra para intercambiarlas.
  4. Puedes Pausar el juego en cualquier momento.
  5. El botón Detener Juego finaliza la partida actual.

Configuración

  • Tamaño de la ficha: Cambia las dimensiones de las fichas (pequeña, mediana o grande).
  • Tamaño del tablero: Define el número de filas y columnas (de 2 a 7).
  • Forma de fichas: Elige entre cuadrados, círculos o triángulos.
